HexDI is a high-performance structural dependency injection engine for TypeScript. Full application topology mapping before execution to eliminate runtime failures.

$ npm i @hex-di/core

CORE_CAPABILITIES

/01_STRICT_VAL

Compile Validation

Detect circular dependencies and missing providers during the build phase. Zero runtime lookup overhead.

/02_SPEED_OPT

Pre-computed Paths

Resolution graphs are pre-mapped, ensuring O(1) instantiation speed for complex dependency trees.

/03_TYPED_CORE

Type Assurance

Native TypeScript inference across all boundaries. No custom decorators required to maintain strict types.

TOPOLOGY_LOGIC

BOOTSTRAP.TS 
import {
            Container } from '@hex-di/core';

            // Define structural services
            class Service { ... }
            class Controller { constructor(s: Service) {} }

            // Secure registry
            const app = new Container()
            .register(Service)
            .register(Controller)
            .build();

            const api = app.resolve(Controller);
