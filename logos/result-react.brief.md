# @hex-di/result-react

> See [LOGO-BRIEF.md](./LOGO-BRIEF.md) for shared brand context, color family, and technical specs.

## What It Is

React bindings for Result types: `<Match>` component for rendering Ok/Err branches, `useResult()` hook, suspense integration.

## Role & Metaphor

**The Result Rendered.** Translates the mathematical Result container into visual React components - the Result becomes something you can see and interact with on screen.

## Key Concepts to Convey

- Result types rendered in React UI
- Component-based branching (Match Ok/Err)
- React integration (orbits/atom motif)
- Visual rendering of success/error states

## Visual Direction

- The Result hexagon (split dual-color) combined with React's orbital/atom motif
- Electron orbits wrapping around the bifurcated hexagon
- Or: the Result hexagon inscribed within React-style orbital ellipses

## Signature Colors

- Base: `#10B981` (emerald) + `#F97316` (orange) from result
- React orbits: `#0EA5E9` (sky blue, React's identity)

## Designer Prompt

> Design a minimal geometric logo icon (200x200 SVG) combining the Result concept (a small hexagon split into green #10B981 and orange #F97316 halves) with React's atomic orbital motif (three intersecting elliptical rings in sky blue #0EA5E9 surrounding the hexagon). The hexagon serves as the atom's nucleus. Orbital ellipses should be thin strokes with small electron dots. Conveys: "Result types integrated into React's component model." No text. Dark background friendly. Must read clearly at 32px.
