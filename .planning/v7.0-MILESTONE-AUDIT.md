## Tech Debt

### Phase 24: Container Instrumentation (1 item)

1. **Dynamic child container auto-instrumentation** — `instrumentContainerTree` walks existing hierarchy correctly but cannot auto-instrument dynamically created child containers. Two structural issues: (a) runtime doesn't emit `child-created` events, (b) InspectorAPI→Container reverse lookup chicken-and-egg problem. Documented as ENH-05 for v8.0.

### Phase 27: Framework Integrations (4 items)

2. **NoOp tracer overhead ~38%** — Target was <5%. Overhead comes from hook invocation machinery (beforeResolve/afterResolve), not the NoOp tracer itself. Documented in benchmark.

3. **Memory tracer overhead ~602%** — Target was <10%. Overhead from span creation, serialization, storage. Expected for full in-memory tracing. Documented in benchmark.

4. **Hono middleware lint warnings** — 7 warnings from Hono Headers API typing limitations. Not actual unsafe code.

5. **Tracing type-guards lint warnings** — 2 `@typescript-eslint/no-unsafe-call` warnings from dynamically accessed crypto/console APIs. Intentional pattern.

### Pre-existing (not from v7.0)

6. **libs/flow/core lint warnings** — 31 `@typescript-eslint/no-unused-vars` warnings. Cleanup in progress.

7. **examples/react-showcase test failures** — 10 test failures from missing localStorage mock. Not related to tracing.

### Total: 5 v7.0 items + 2 pre-existing items
