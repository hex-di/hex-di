## Goal

Create test assertion helpers and span matchers for verifying tracing behavior in tests, fulfilling TEST-02 and TEST-03 requirements.

## Tasks

<task>
Create the testing utilities directory and implement assertSpanExists with SpanMatcher interface:
- Create packages/tracing/src/testing/ directory structure
- Implement assertSpanExists function that finds spans matching criteria
- Support name matching with string or RegExp
- Support status, attributes, hasEvent, minDuration matching
- Throw descriptive error if no match found with details about what was searched
- Ensure pure functions with no side effects
</task>

<task>
Implement span matcher predicates for common assertions:
- Create hasAttribute(span, key, value?) predicate for attribute checking
- Create hasEvent(span, name) predicate for event presence
- Create hasStatus(span, status) predicate for status matching
- Create hasDuration(span, minMs?, maxMs?) predicate for timing checks
- All matchers must be pure functions returning boolean
- Support optional value matching for hasAttribute
</task>

<task>
Create testing index exports and update main package exports:
- Create packages/tracing/src/testing/index.ts exporting all utilities
- Update packages/tracing/src/index.ts to export from "./testing/index.js"
- Ensure tree-shaking friendly exports with separate entry point
- Add JSDoc documentation for all exported functions
</task>

<task>
Write comprehensive unit tests for assertion helpers:
- Test assertSpanExists with various matchers (name, status, attributes)
- Test RegExp name matching and partial attribute matching
- Test error messages when no span matches
- Test empty spans array edge case
- Verify all matchers work correctly with edge cases
- Test that functions are pure (no mutations)
</task>
