    parseTraceparent(header: string) function:
    - Split by '-' expecting 4 parts
    - Validate version === '00'
    - Validate traceId: 32 hex chars, not all zeros
    - Validate spanId: 16 hex chars, not all zeros
    - Validate flags: 2 hex chars

    - Format as: 00-{traceId}-{spanId}-{flags}
    - flags as 2-char hex from traceFlags number

    isValidTraceId(id: string) function:
    - Check 32 hex chars via regex
    - Check not all zeros

    isValidSpanId(id: string) function:
    - Check 16 hex chars via regex
    - Check not all zeros

    Add JSDoc with W3C spec references.
    NO type casts in validation.
