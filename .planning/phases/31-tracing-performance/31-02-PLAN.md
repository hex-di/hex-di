    2. Create hex lookup table for fast conversion:
       - `const HEX_CHARS = "0123456789abcdef".split("");`

    3. Implement bytesToHex function:
       ```typescript
       function bytesToHex(bytes: Uint8Array): string {
         let hex = "";
         for (let i = 0; i < bytes.length; i++) {
           const byte = bytes[i]!;
           hex += HEX_CHARS[byte >> 4]! + HEX_CHARS[byte & 0xf]!;
         }
         return hex;
       }
       ```

    4. Update generateTraceId and generateSpanId:
       - Use crypto.getRandomValues(buffer)
       - Check for all-zeros (invalid per spec)
       - Return bytesToHex(buffer)
       - Keep fallback for environments without crypto
