## Tech Debt

### Phase 04: API Ergonomics

**Item:** `fromClass().as<Interface>('Name')` interface narrowing doesn't work

**Details:** The `.as<Interface>()` syntax requires constraint `TService extends TInstance`, but in TypeScript `Logger extends ConsoleLogger` is false even when `ConsoleLogger implements Logger`. This is a TypeScript limitation.

**Workaround:** Use `.as('Name')` without type parameter. Service type inferred from class instance.

**Impact:** Minor - affects only interface narrowing use case, core functionality works.

**Recommendation:** Document limitation, address in future iteration if solution found.
