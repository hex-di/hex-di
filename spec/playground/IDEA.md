# HexDi Playground

## Raw Idea

HexDi Playground - an interactive web-based playground for experimenting with HexDi dependency injection patterns in the browser. Should share significant infrastructure with HexDi Devtools (graph visualization, panel system, theme system, etc.) but focused on code experimentation rather than runtime inspection.

Key context: The devtools spec is complete at `spec/devtools/` with 8 documents covering a standalone web dashboard for inspecting HexDi applications. The playground should leverage shared components like the graph renderer, panel architecture, theme tokens, and potentially the panel plugin system.

Think of it like TypeScript Playground meets the devtools visualization capabilities.
