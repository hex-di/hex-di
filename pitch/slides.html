HexDIThe Problem

What HexDI is

The tooling layer that turns Hexagonal Architecture from a convention into a structural constraint.

Boundaries enforced

  • Every external dependency is declared as a typed port
  • There is no mechanism to bypass a port — the factory only receives what is declared
  • Violations are compile errors, not code review comments

Architecture live

  • The dependency graph is a real object generated from the running code
  • Always accurate — it is the code, not a diagram
  • Queryable, exportable, AI-readable

Tools swappable in one line

  • Change Pino to Winston: one line in the GraphBuilder
  • The compiler confirms every consumer is satisfied
  • Business logic is completely untouched

Keep your existing tools

  • HexDI does not replace Pino, React Query, or XState
  • It gives them clean boundaries via adapters
  • Your team keeps what it knows — with enforced architecture around it

HexDILiving Architecture

The AI problem has a structural solution

AI tools are pattern-matchers. In an unstructured codebase, they match bad patterns too.

Without HexDI

  • AI infers architecture from the files it sees — including the exceptions
  • AI-generated code bypasses ports silently — no mechanism to prevent it
  • Review burden explodes — humans must catch every structural mistake
  • METR 2025: experienced developers are 19% slower with AI tools — because review takes longer than writing

The root cause

  • AI generates fast. Structural validation is slow and human.
  • The bottleneck is not writing code — it is knowing whether the code is correct.

With HexDI

  • Every boundary is explicit — the AI sees typed ports, not implicit imports
  • The MCP server gives AI live access to the graph — it knows valid wiring before generating
  • The compiler validates AI output — wrong boundaries are compile errors before commit
  • Code review focuses on business logic — structural correctness is handled by the machine
  • AI generates consistent architecture because the structure is explicit and enforced
