# Welcome to the HexDI Blog

We're excited to launch the HexDI engineering blog! This is where we'll share architecture insights, TypeScript deep dives, and updates on the HexDI ecosystem.

<!-- truncate -->

## What to Expect

Here's what we'll be writing about:

- **Architecture decisions** — The reasoning behind HexDI's design, from hexagonal architecture to compile-time dependency validation
- **TypeScript type system** — Advanced techniques we use to catch dependency errors at compile time instead of runtime
- **Tutorials** — Step-by-step guides for building applications with HexDI
- **Release notes** — What's new in each version, migration guides, and roadmap updates

## Get Involved

HexDI is open source and we welcome contributions. Whether it's code, documentation, or blog posts, there are many ways to get involved:

- Browse the [source code](https://github.com/leaderiop/hex-di)
- Read the [documentation](/docs/getting-started)
- Join the [discussions](https://github.com/leaderiop/hex-di/discussions)

Stay tuned for more posts!
