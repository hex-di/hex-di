# Raw Idea

Improve HexDiDevtools to be a wrapper for DevtoolsPlugins like GraphDevtools/ServicesDevtools/TracingDevtools/InspectorDevTools. Each plugin should have its own pane (Graph | Services | Tracing | Inspector). The Devtools provides the containers graph to plugins. Users can create custom devtools plugins, for example MCP/A2A panes or a chat for self-improvement.
